# Data Frame of all fixed discrete palettes

A data.frame of the names and other characteristics of the discrete
palettes of fixed lengths included in this package.

## Usage

``` r
palettes_d_names
```

## Format

A data.frame of 2484 observations with 4 variables:

- package:

  character, name of package

- palette:

  character, name of palette

- length:

  integer, number of colors in palette

- type:

  character, type of palette
